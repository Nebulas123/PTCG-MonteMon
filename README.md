# PTCG-MonteMon

Monte Carlo simulator for estimating whether a Pokemon TCG opening can reach a target board state within the first two turns.

## Goal

PTCG-MonteMon is intended to help test deck setup consistency:

- model opening hands, prizes, draws, and searches
- simulate early-turn play patterns
- estimate the probability of reaching a chosen board state by turn two
- compare card counts and route choices across deck lists
